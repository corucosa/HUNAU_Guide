# 非机动车出行

## 校内骑行
农大校内有数量庞大的美团单车投放，可以通过nfc碰一碰和扫码骑行。推荐购买月卡，长沙市范围内有效。但请注意破损故障车也很多，所以正式骑行前得试一试（开锁后一分钟内取消本次骑行免费）。

你也可以购买一辆属于自己的自行车，原则上不允许学生在校内骑电动车。

虽然校内治安尚可，但记得看好你的车！

## 校外骑行
长沙的路面状况非常不尽如人意：路面破损和起伏严重、没有独立的非机动车道（小心撞大运）。骑车多注意安全。

长沙城区治安状况一般，请时刻记得离车的时候就上锁，谨防被盗。

## btw，校外去哪骑？
- 南门出去的浏阳河绿道，路面连续完整，风景不错，但是小心路上的人；
- 南门出去的滨河路（不上堤上的绿道），路面状况尚可，但小心机动车；
- 东门出去的东湖公园，很夯，就是蚊子多，以及长度不够骑（哭）；
- 星沙的松雅湖，有点远，但是是骑行天堂。
