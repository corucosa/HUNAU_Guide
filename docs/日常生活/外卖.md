# 外卖

## 可以点哪些外卖？
如果你不想临时骑车到校门口（特别是大晚上的时候），请一定要选择有“配送进校”字样的外卖，学校食堂的大部分档口和部分校外商家都可以配送进校

对于芷兰六这种离门近的寝室楼，也可以考虑选择不配送进校的，这种情况下，外卖员会将外卖送至铁门处

## 外卖送到哪？
外卖只允许放在外卖架上。宿舍的外卖架在自己的宿舍园区外，每栋教学楼则都有外卖架。

外卖到了就及时拿，小心被偷（不嘻嘻:（

校园送晚上十点之后就不送了。

## 如果没有配送进校呢？
外卖会放在学校大门，如果没写哪个门，一般外卖员会和你打电话沟通，或者直接放在北门。

强烈建议不要把外卖地址留在北门，北门是偷外卖的高发地。

## 外卖被偷怎么办？
先联系骑手，很多时候是骑手为了赶时间提前点送达了。但如果骑手确认他放到架子上了，那就及时联系商家，一般会退款/再送一份/下次点多给点xx。

## 乱说点
农大附近有很多店是专做学生生意的外卖店，没有堂食。点餐之前可以自行留意一下，其中有些店铺被抖音的政府账号 @长沙市场监督 点名“表扬”过。
