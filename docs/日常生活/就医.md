# 就医

## 校医院在哪？
校医院在学校南门外，出门右手边，但只能够处理一些简单的故障。

## 学校周边的大医院呢？
- 最近的：长沙市第八医院（长沙市中医院），长沙县的是本部，其它地方的是分部，可以在西门坐X111路公交车到第八医院下车。
- 湖南省旺旺医院，旺旺集团（就是你想的那个旺旺）的民营医院，服务态度好，价格相对实惠，但只属于三级医院而非三甲医院。坐地铁6号线到芙蓉区政府站下车。
- 湖南知名的医院们：湘雅医院、湘雅二院，可以坐6号线直达，分别在湘雅医院站和湘雅二院站下车。

## 关于医保
可以依然继续缴纳自己以前所在地的医保，也可以跟着学校一起缴纳大学生医保。大学生医保能够直接在校医院内报销使用，在长沙市范围内享受待遇与城乡居民医保一致。也可以自己缴纳长沙市普通居民医保。如果先前在长沙市以外缴纳的医保，可以去搜索你们当地的医保如何转出，以及如何将医保转入长沙市。

如果缴纳的是长沙市以外的医保，建议在开学时就申请异地医保备案申请，可以用自己的学生证/录取通知书等在支付宝和微信申请，一般批准的期限是半年至一年不等，以备不时之需（但祝你不会有这需求：）。

## 医保能报销多少？
长沙的居民医保的门诊报销（挂号、拍片、买药），只能在基层医疗机构（比如校医院、）报销。如果去八院湘雅旺旺的话，你要全额支付这些费用。

每年可以在基层医疗机构一共报销560块钱，每次可以报销70%，你只要掏30%，相当于打三折。但560块钱花完就没了。
